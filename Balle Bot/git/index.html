Balle Bot

Deixe a sua comunidade segura de um jeito fácil!

o que ela faz por você?

1- cria um sistema capcha para novos membros garantirem que não são robôs.

2- Possui um sistema de banimento automatico para links suspeitos que forem enviados para na comunidade.

3- sistema de aviso e banimento manual para pessoas administradoras utilizarem.
